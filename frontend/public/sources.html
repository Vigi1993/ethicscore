Tier 3 includes blogs, specialist publications, and secondary sources. Used only as supporting evidence alongside higher-tier sources — never as the sole basis for a score. Evaluated case-by-case, no fixed whitelist.
How we score brands

Each brand is evaluated on 20 criteria across 4 categories. Each criterion receives an absolute score based on the tier of available sources. The total score ranges from -400 (worst) to +400 (best). A criterion only contributes if it meets the minimum source threshold.

Minimum threshold per criterion
T1  At least 1 Tier 1 source → uses the average of all T1 sources
T2  No T1, at least 2 Tier 2 sources → uses the average of T2 sources
T3  No T1, 1 T2 + at least 3 T3 → T2 value + average of T3 (summed, capped at ±20)
